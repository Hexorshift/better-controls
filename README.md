# 🕹️ Better Controls

✈️ Intuitively control flight surfaces with your mouse in Microsoft Flight Simulator or any other simulator.

## 🔹 Overview

Better Controls lets you use your mouse to control flight surfaces such as ailerons, elevators, and rudder in Microsoft Flight Simulator (MSFS). It also provides hotkeys and a compact HUD overlay to show current control state.

## 🌟 Features

- 🖱️ **Mouse Yoke:** Move your mouse to control pitch and roll.
- 🎛️ **Rudder Control:** Hold left mouse button to control rudder.
- 🔄 **Invert Elevator:** Optionally invert the elevator axis.
- 🖥️ **Compact HUD:** Shows controller, elevator, and rudder status (only visible when MSFS is active).
- ⌨️ **Hotkeys:**
  - `Ctrl + F1` – Toggle controller on/off
  - `Ctrl + F2` – Invert elevator
  - `Ctrl + F3` – Center mouse

## 🛠️ Installation

1. Download the latest `.exe` release.
2. Ensure the [ViGEmBus driver](https://vigem.org/projects/ViGEm/) is installed for virtual controller support.
3. Run `Better Controls.exe`.

## 🚀 Usage

1. Launch Microsoft Flight Simulator.
2. Run `Better Controls.exe`.
3. Use the hotkeys to toggle the controller, invert elevator, or center the mouse.
4. Move your mouse to control the yoke.
5. Hold the left mouse button to activate rudder control.

## 📋 Requirements

- Windows 10 or 11
- ViGEmBus driver installed

## ⚠️ Troubleshooting

- ❌ **HUD not visible:** Make sure MSFS is the active window and controller is enabled.
- ❌ **Controller not responding:** Verify ViGEmBus is installed and running.
- ❌ **Hotkeys not working:** Ensure MSFS is not using the same shortcuts.
